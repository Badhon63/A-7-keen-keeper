# KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a web application that helps users maintain meaningful relationships by tracking interactions with friends and reminding them to stay connected.

---

## 🚀 Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- DaisyUI
- Recharts
- React Toastify

---

## ✨ Key Features

1. Track and manage friendships with detailed friend profiles and status indicators  
2. Log interactions (Call, Text, Video) and view them in a timeline with filtering  
3. Visualize interaction data using a responsive pie chart on the analytics page
